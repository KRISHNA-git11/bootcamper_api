# BootCamper API

> Backend API for any Bootcamps organisation application

## Usage

Reanme "config/config.env.env" file to "config/config.env" and update the values/settings to your own credentials

## Install Dependencies

```
npm install
```

## Run App


## Run in dev mode
```
npm run dev
```
## Run in prod mode
```
npm start
```
